# 🎯 Quick Start Guide - Assistente IA Melhorado

## Como Usar o Sistema de IA Redesenhado

### 1️⃣ Acessar o Chat
```bash
# Via navegador, acesse:
http://localhost:5000/chat

# Ou clique em:
Sidebar → Assistente IA
```

### 2️⃣ Fazer uma Pergunta

#### Opção A: Digitar Manualmente
1. Clique no campo de entrada
2. Digite sua pergunta (ex: "Quanto vendi hoje?")
3. Pressione Enter ou clique no ícone de envio ➤

#### Opção B: Usar Sugestões Rápidas
1. Veja as 4 sugestões no topo (ícones com cores)
2. Clique em qualquer uma
3. A pergunta aparecerá no input
4. Pressione Enter para enviar

---

## 🆕 Novas Features

### 📋 Limpar Conversa

**Como funciona:**
1. Clique no botão **🗑️ Limpar** (lado direito da barra de sugestões)
2. Apareçerá um aviso: "Tem certeza que deseja limpar toda a conversa?"
3. Clique em **OK** para confirmar
4. Chat será limpo e volta ao estado vazio

⚠️ **Cuidado**: Esta ação é irreversível!

---

### 💡 Destaque de Insights

**O que é:**
Números e valores importantes aparecem destacados com cor amarela.

**Exemplos:**
- "Você vendeu **R$ 5.234,50** hoje" ← destaque em amarelo
- "Você tem **12 produtos** em estoque baixo" ← destaque em amarelo

---

### 📊 Respostas Estruturadas

O sistema formata automaticamente respostas complexas.

#### Exemplo 1: Listas
**Pergunta**: "Quais são meus 3 melhores produtos?"

**Resposta estruturada:**
```
✓ Produto A - R$ 3.500
✓ Produto B - R$ 2.800  
✓ Produto C - R$ 1.900
```

#### Exemplo 2: Numeradas
**Pergunta**: "Quais ações devo tomar?"

**Resposta estruturada:**
```
① Reduzir estoque do produto X
② Aumentar marketing para produto Y
③ Buscar novo fornecedor
```

## 🌙 Dark Mode

**Como ativar/desativar:**
1. Clique no ícone de **lua/sol** no header
2. Chat mudará para dark mode (ou light mode)
3. Sua preferência é salva automaticamente

**Cores:**
- Light Mode: Fundo claro, texto escuro
- Dark Mode: Fundo escuro, texto claro

## ❌ Troubleshooting

### Problema: Chat não carrega
**Solução:**
1. Atualize a página (F5)
2. Limpe o cache (Ctrl+Shift+Delete)
3. Faça login novamente

### Problema: Sugestões não aparecem
**Solução:**
1. Aguarde alguns segundos
2. Atualize a página
3. Verifique sua conexão

### Problema: Resposta muito lenta
**Solução:**
1. Verifique sua conexão de internet
2. Tente uma pergunta mais simples
3. Contacte o administrador

### Problema: Botão Limpar não funciona
**Solução:**
1. Verifique se está logado
2. Tente novamente
3. Se persistir, faça logout e login novamente
